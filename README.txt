# Problemas de Lógica en JavaScript

Este repositorio contiene soluciones en JavaScript a diferentes problemas de lógica sacados de LeetCode. Cada archivo contiene una solución algorítmica para un problema específico, junto con su descripción y enfoque.

## Estructura del Repositorio

El repositorio se organiza en carpetas, donde cada carpeta representa un problema en particular. Cada carpeta contiene los siguientes elementos:

- **`problem-description.md`**: Un archivo markdown que proporciona la descripción del problema, incluyendo el enunciado y los ejemplos de entrada y salida.

- **`solution.js`**: Un archivo JavaScript que contiene la solución al problema.


## Contribuciones

Siéntete libre de contribuir a este repositorio añadiendo nuevas soluciones o mejorando las existentes. Asegúrate de seguir las siguientes pautas al contribuir:

1. Crea una nueva rama para tu contribución: `git checkout -b nombre-de-la-rama`.

2. Asegúrate de que tu código sea legible y esté correctamente documentado.

3. Incluye comentarios que expliquen tu enfoque y cualquier suposición o limitación relevante.

4. Agrega tu solución al problema correspondiente y asegúrate de que funcione correctamente.

5. Envía una solicitud de extracción (pull request) una vez que hayas completado tu contribución.

¡Espero que encuentres útil este repositorio y que te diviertas resolviendo los problemas de lógica! Si tienes alguna pregunta o sugerencia, no dudes en abrir un problema (issue) en el repositorio.

Happy coding! 😄
